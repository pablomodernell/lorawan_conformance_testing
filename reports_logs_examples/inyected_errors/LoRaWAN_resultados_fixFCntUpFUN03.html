Agent configuration tutorial.

1-Configure the Packet Forwarder on the LoRa Gateway.: e.g. Agent UDP port (default 1700) and the host IP on gateway's local.conf file

2-Set AMQP Broker URL: e.g.: export AMQP_URL=amqp://guest:guest@localhost:5672/

3-Set Packet Forwarder UDP PORT (LoRa Gateway UDP Port): e.g.: export AGENT_PORT=1700

4-Start Agent Service: make start_agent
